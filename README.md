# pthread_example
It is a example of pthread fork() and POSIX shared memory

Implementing pthread fork() and POSIX shared memory in a 1A2B game.


- The used function from pthread
  - fork()
  - getpid() 
  
- The used function from POSIX shared memory
  - shm_open()
  - ftruncate()
  - mmap
  - munmap
  - shm_unlink()
